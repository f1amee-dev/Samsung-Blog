# Iza razglednice

Studentski portfolio o turizmu, mladima i budućnosti Istre, rađen za Samsung Izazov.

Petero nas iz 1.a u Gimnaziji i strukovnoj školi Jurja Dobrile u Pazinu: Filip, Eva, Jura, Luana i Alen. Mentori su nam profesor Lovro Šverko i pedagoginja Iva Luk.

Dva tjedna, sedam razgovora. Pričali smo s turističkom zajednicom, obrtničkom komorom, Valamarom, gradonačelnicom Pazina, AZRRI-jem, Koradom Korlevićem i Uslugom Poreč, pa pokušali složiti što nam je od svega ostalo u glavi.

Stranica je napravljena u Astru, tekstovi su u `src/content/blog`.

## Pokretanje

```sh
bun install
bun dev
```

Build ide u `dist/` preko `bun run build`.
